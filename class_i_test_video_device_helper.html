Video device testing callback event.

Remarks
This function must be called before using any other interface in this class.
Parameters
pEvent[in] A pointer to an ITestVideoDeviceHelperEvent that receives video device testing event.
Returns
If the function succeeds, the return value is SDKErr_Success. If the function fails, the return value is not SDKErr_Success. To get extended error information, refer to SDKError enum.

pure virtual

Start to test the camera.

Parameters
deviceID[in] Specifies a camera device to test. If the param is a wrong camera ID, SDK returns an error. Otherwise SDK tests the specified device and sets it to be the selected one. If no param is passed in, SDK tests the default device.
Remarks
The function can not work if no event is set or no window handle is set.
