◆ SDK_TESTMIC_STATUS

enum SDK_TESTMIC_STATUS

Notify the status of the mic during the mic testing.

Enumerator
enuCanTest 

Can call TestMicStartRecording to start testing mic. Useless to call TestMicStopTesting/TestMicPlayRecording.

enuMicRecording 

Can call TestMicStopTesting/TestMicPlayRecording. Useless to call TestMicStartRecording.

enuCanPlay 

Can call TestMicStopTesting/TestMicPlayRecording. Useless to call TestMicStartRecording. enuCanPlay status means SDK has recorded the mic sound for the longest time (6 seconds). In this status, if the bHandled is not set to be 'true', SDK will call TestMicStopTesting itself.
